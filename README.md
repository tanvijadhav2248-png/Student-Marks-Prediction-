# 🎯 Student Marks Prediction System

## 📌 Project Overview

This project predicts student marks based on:

* Study Hours 📚
* Attendance Percentage 🏫

It uses **Machine Learning (Linear Regression)** to generate accurate predictions.

---

## 🚀 Features

* Predict marks based on user input
* Model evaluation using:

  * Mean Absolute Error (MAE)
  * R2 Score
* Data visualization using matplotlib
* Simple command-line interface

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* Scikit-learn
* Matplotlib

---

## 📊 Sample Output

Input:
Study Hours = 5
Attendance = 80

Output:
Predicted Marks ≈ 60
